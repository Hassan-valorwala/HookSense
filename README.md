# HookSense — AI Content Decision Tool

HookSense helps small business owners compare draft Instagram Reels and decide which version to post — using AI analysis of hook strength, pacing, and audio energy, with plain-English explanations.

## Tech Stack
- **Frontend**: React.js + Tailwind CSS + Vite
- **Backend**: Python + FastAPI
- **Video Processing**: OpenCV
- **Audio Processing**: librosa
- **AI Explanation**: Groq API (LLaMA 3)
